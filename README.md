# IT-helpdesk-chatbot
Python-based IT Helpdesk Chatbot that provides automated responses to common user issues like password reset, internet connectivity, and system troubleshooting using a simple GUI.

📌 Overview
This project is a Python-based IT Helpdesk Chatbot that provides automated responses to common technical issues such as password reset, internet problems, VPN errors, and system performance. It uses a simple GUI built with Tkinter to simulate real-time customer support interaction.

🚀 Features
	•	Handles common IT support queries
	•	Provides instant automated responses
	•	Simple and user-friendly GUI
	•	Rule-based chatbot logic
	•	Real-time chat interface

🛠️ Technologies Used
	•	Python
	•	Tkinter

▶️ How to Run
	1.	Install Python
	2.	Download/clone the repository
  3.  Run the file:
      helpdesk_chatbot.py
      
🎯 Purpose
This project demonstrates basic chatbot development and simulates an IT helpdesk environment, helping improve problem-solving and technical support skills.
