# IT-helpdesk-chatbot
Python-based IT Helpdesk Chatbot that provides automated responses to common user issues like password reset, internet connectivity, and system troubleshooting using a simple GUI.
This project is a Python-based IT Helpdesk Chatbot built using Tkinter. It simulates real-time customer support by responding to common technical issues such as password reset, internet problems, VPN, and system performance. The chatbot uses rule-based logic to provide quick and accurate solutions, improving user experience.
